After creating the view, I had to go into access and link the view to the database which can easily be done by following these steps:

1.       On the top menu go to the tab External Data

2.       Click on ODBC Database

3.       On the next screen create a new ODBC Connection and give it a name.

4.       After clicking New that would bring up a window asking you to specify the type of the Data Source.
